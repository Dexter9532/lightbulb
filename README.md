# Lightbulb

Lightbulb is a focused mobile app for turning ideas into projects.

It lets you:
- create ideas
- add brainstorm notes
- attach components
- organize segments with descriptions, links, and references
- use simple idea templates
- drag ideas into projects on a dashboard
- move, archive, or remove items freely

Nothing extra is included beyond that scope.

## Stack

- React + TypeScript
- Capacitor Android shell
- Local device storage for persistence

## Run locally

### Web preview

```bash
npm install
npm run dev
```

### Sync the Android project

```bash
npm install
npm run cap:sync
```

### Open in Android Studio

```bash
npm run android:open
```

## APK

The repo is set up so tagged releases build an APK in GitHub Actions.

### Download the current APK

1. Open the repo's **Releases** page on GitHub.
2. Download the attached `lightbulb-<tag>-debug.apk` from the latest release.
3. Install it on your Android device.

## Create a new release tag

Each app version should be tied to a Git tag.

Example:

```bash
git tag v0.1.0
git push origin v0.1.0
```

When that tag is pushed, GitHub Actions will:
- build the app
- generate the Android APK
- attach the APK to the GitHub release for that tag

## Notes

- The APK generated by the release workflow is a debug APK unless the signing setup is expanded later.
- App data is stored locally on the device.
